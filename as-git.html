Google最近发布了Android Studio 1.0正式版，作为一个Android开发者不能不紧跟技术潮流，废话少说，本文将介绍在Android Studio上配置Git，并通过Git上传代码。

准备工作

  1. Android Studio ,本文使用的是当时最新版 Android Studio v1.0.1

  2. Git for Windows

    安装完AS和Git Gui工具后，在AS上设置Git路径

    File > Settings > Version Control > Git

第一步：开启git

VCS > Enable Version Control Integration > select "Git"

开启git之后，文件变成红色。

第二步：添加远程库

这一步不能通过AS完成，AS目前没有提供这个功能，所以我们只能通过Git Base来操作。

  1. 打开Git Base，进入项目目录

  2. Git Base > 输入添加远程库命令

    前提是已经创建了远程库，而已库必须干净，不能包含任何文件包括readme文件，这里已国内osc git为例。

git remote add origin https://git.oschina.net/xxxx/ASGit.git

第三步：添加项目到git 版本控制

首先 VCS > Git > Add 成功添加后文件变成绿色。

第四步：commit到git 服务器
VCS > Commit Changes...

若选择commit and push将跳到第五步的操作

第五步：push到git 服务器

VCS >Git > Push...

勾选分支，点击push

至此，已经提交成功！可以做其他的一些操作，如commit，create branch等。
